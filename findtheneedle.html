Game Instructions

This is a card guessing game. You start with 8 guesses. Each game costs £100. If you find the Joker, you win and your reward is multiplied based on the stage you are in:

  • Stage 1: 0.5x money back
  • Stage 2: 1x money back
  • Stage 3: 2x money back
  • Stage 4: 3x money back
  • And so on...

If you run out of guesses, the game resets.
